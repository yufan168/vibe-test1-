#SUSU 專用工作台
這個 repo是我（SUSU）和蝦思的「日常工作流程庫」。
目標：把重複工作標準化，減少忘記與重工。
#＃這裡放什麼
-日曆建立與提醒流程
-雲端硬碟整理流程
- 考試/活動報名流程
- 可重複使用的模板
#＃快速開始（3步驟）
1. 先看‘docs/calendar-guide.md'
2. 依模板填寫需求（時間、地點、提醒）
3. 交給蝦思執行並回報結果
#＃目錄
- 'docs/：操作說明與 SOP
- 'templates/：可直接套用的格式
-'examples/：範例輸入與輸出
- 'skills/：可擴充的技能說明
##使用原則
-同一件事盡量用同一格式
- 重要流程要可複製、可追蹤
- 每次調整都更新文件，避免口頭規則遺失
