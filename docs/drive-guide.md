# Drive Guide (雲端硬理SOP)
##目標
將檔案可搜尋、可分享、可回收,不亂丢不重複。
##整理前先確認
-整理範圍(整個雲端or指定資料夾)
-分類方式(主題/日期/專案)
-命名規則(YYYYYMMDD_主題_版本)
-重複檔規則(先標記,不先刪除)
##建議流程
1.先盤點清單
2.再做分類搬移
3.再做命名統一
4.最後處理重複檔(先進待確認刪除)
##分享規則
-需要分享時才開連結
-重要檔案使用固定資料夾
-連結建立後要測試可開取
##安全規則
-不直接刪除,先移到「待確認刪除」
-每次大整理前先備份
-保留變更紀錄(日期/異動内容)
##常用口令
-先盤點,不刪除
- 先分類與命名
- 產生分享連結
-把重複檔移到待確認刪除
